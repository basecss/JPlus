新建项目

要加入J+，就需要先在J+中创建一个项目。本文将指导您如何新建项目。

第一步、取一个英文项目名(可以使用数字、下划线，但不能有空格和其它特殊符号)

本页面假设项目名为weued 。

第二步、在 assets/project/ 新建 weued.js

在这里定义项目的导航页。其内容比如： 

var menus = {
				
	'菜单类目1': {
		'子菜单1': '点击子菜单1后到的链接',
		'子菜单2': '点击子菜单2后到的链接'	
	},
	
	'菜单类目2': {
		'子菜单3': '点击子菜单3后到的链接'
	}	
			
};

第二步、在 src/ 新建 weued.js

这个文件用于定义您的项目的基础文件，所有用到您的项目下的代码时，都必须先引入此文件。默认情况下，此为空文件。

第四步、在跟目录新建一个 weued 文件夹。

此文件夹存放您项目的所有文件。

第五步、在 assets/project/project.js 找到 navs 对象，添加新的导航菜单。

到此，项目已初始化完成。
