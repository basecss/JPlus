Index

Core 模块是全部DPL共用的 JS 和 CSS

文本链接颜色

  • 默认: 标准链接
  • b-hint: 辅助链接
  • b-hignlight: 重要链接
  • b-alt: 次要链接

文本颜色

  • 默认: 正文
  • b-hint: 辅助
  • b-hignlight: 强调

公共样式

  • b-left: 左浮动
  • b-center: 居中
  • b-right: 右浮动
  • b-clear: 清除浮动
  • b-inline: 内联
  • b-block: 块
  • b-inline-block: 内联块
  • b-hide: 影藏元素（display:none)
  • b-relative: 相对定位
  • b-absolute: 绝对定位
  • b-blank: 空行（10像素高）
  • b-break-word: 强制文本换行，避免长单词不换行
  • b-content: 正文
  • b-menu: 菜单（菜单内的 ul 和 li 将无边距）
  • b-debug: 高亮
