

Git  使用方法 



一、安装GIT


1. 安装 GIT 插件    。


http://download.eclipse.org/egit/updates




2. 选择文件/导入/Git/Projects From Git... 。


3. 点击右边的复制... 。



4.  输入 URI (https://用户名@github.com/weued/JPlus.git) 。



5. 输入其它信息， 下一步。


6. 选择 master 分支。


7. 设置本地位置。


8. 按提示完成。



二、配置 GIT 。



1.  打开 .git/config  (  用 eclipse )
2.  底部加上



[branch "master"]
    remote = master
    merge = refs/heads/master




三、使用 GIT 。




提交 - Commit - 在本地提交更改
拉   - Pull   - 从服务器获取最新的版本
推   - Push   - 将本地提交的代码更新到服务器
撤销 - Revert - 撤销本地的提交，文件会恢复为服务器版本。
